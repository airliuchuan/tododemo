#### 1. 技术栈

由 [create-react-app]构建webpack环境 + 使用[less]css预处理器 使用 [redux]和[react-redux] 进行状态管理


#### 2.Build Setup


	# 安装项目所需模块
	npm install

	# 运行在localhost:3000
	npm start

	# 生产环境打包
	npm run build

	# 弹射(用于修改webpack配置, 不可逆)
	npm run eject
